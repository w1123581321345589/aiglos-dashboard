Documentation

Aiglos is an in-process MCP security runtime. It intercepts every tool call your AI agents make — before execution — and inspects it against 36 rule families in under a millisecond.

Installation

Aiglos requires Python 3.9+. Install from PyPI:

SHELL 
pip install aiglos

For TypeScript / Node.js:

SHELL 
npm install aiglos

Quickstart

Add one import before your agent starts. Aiglos patches mcp.client.session.ClientSession at import time. No config file. No proxy server. No port.

PYTHON 
import aiglos  # add this line — that's it

# your existing agent code below — nothing else changes
from mcp import ClientSession
from langchain_mcp import MCPToolkit

# aiglos is now intercepting every tool call
toolkit = MCPToolkit(session=session)
agent = create_react_agent(llm, toolkit.get_tools())
Free tier: With no API key set, Aiglos runs in local-only mode — full T1–T36 detection, zero telemetry, no call limits. This works forever without a key.

To enable cloud telemetry and attestation artifacts, set your API key:

SHELL 
export AIGLOS_KEY=ak_live_pro_your_key_here

How It Works

When you import aiglos, it monkey-patches mcp.client.session.ClientSession.call_tool. Every subsequent tool call is intercepted synchronously before the MCP transport layer sends it.

The fast-path scanner runs 36 rule families against the tool name and arguments. The entire scan takes less than 1ms. Your agent never notices the overhead.

Environment Variables

SHELL 
# Required for paid features
export AIGLOS_KEY=ak_live_pro_...

# Optional
export AIGLOS_MODE=block       # block | warn | audit
export AIGLOS_LOG_LEVEL=info   # debug | info | warning | error

Scan Modes

Aiglos supports three scan modes set via AIGLOS_MODE:

  • block (default) — Raises AiglosBlockedError on BLOCK verdicts, stops tool execution.
  • warn — Logs findings but allows all calls through. Useful during onboarding.
  • audit — Records everything silently. Zero runtime impact. Good for baselining.

Tiers & Gates

Feature access is determined by your API key prefix:

  • ak_live_pro_* — Developer Pro
  • ak_live_team_* — Team
  • ak_live_ent_* — Enterprise
  • No key — Free tier (local only)

The first time a free-tier installation calls a gated feature (aiglos.attest() or cloud telemetry), a 30-day Pro trial starts automatically. A non-blocking prompt appears in stderr. After 30 days, gated features lock until a key is set.

Attestation

Attestation artifacts are RSA-2048 signed JSON documents proving that an Aiglos runtime was active during a given agent session and that every tool call was inspected before execution.

PYTHON 
import aiglos

# At the end of an agent session:
artifact = aiglos.attest({
    "session_id": "sess-abc123",
    "agent_id":   "my-agent",
})

print(artifact["artifact_id"])
print(artifact["session_summary"])

# Verify any artifact offline:
valid = aiglos.verify(artifact)
print(valid)  # True
Verification is free. aiglos.verify() works on any tier with no API key. Only generation requires Pro.

Verdicts

VerdictMeaningAction
CLEAN Passes all 36 rule families Executes normally. No overhead.
WARN Suspicious pattern detected Logged. Call proceeds. Appears in telemetry.
BLOCK Attack pattern confirmed Raises AiglosBlockedError before execution.

Telemetry

With a Pro key, Aiglos batches usage events and sends them to api.aiglos.dev/v1/events every 30 seconds. Telemetry is strictly best-effort — network failures are swallowed silently and never block the agent call path.

The free tier generates zero telemetry. Nothing is sent to any remote endpoint without an API key.

Rule Families

All 36 rule families are MIT licensed. View source on GitHub →

  • T01–T06 — Injection & Exfiltration
  • T07–T12 — Shell & OS Abuse
  • T13–T18 — Network & SSRF
  • T19–T24 — Secrets & Credentials
  • T25–T29 — A2A Protocol
  • T30–T34 — Supply Chain
  • T35 — Personal Agent Monitor
  • T36 — Embedded Mode

Exceptions

When a tool call is blocked, Aiglos raises AiglosBlockedError:

PYTHON 
from aiglos import AiglosBlockedError

try:
    result = await session.call_tool("shell.execute", {"cmd": "rm -rf /"})
except AiglosBlockedError as e:
    print(e.tool_name)          # shell.execute
    print(e.result.reason)      # shell_injection detected
    print(e.result.risk_type)   # T07
