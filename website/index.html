How it works

One import. Full runtime coverage.

Aiglos sits between your agent and every tool call it makes. No code changes required beyond the import.

01
Wrap the session
Pass your MCP session to aiglos.attest(). One line. Zero configuration required.
pip install aiglos
02
Intercept at the wire
Aiglos intercepts every outbound tool call at the MCP transport layer before it reaches the tool server.

Integration

As simple as it should be.

Drop into any agent framework. LangChain, LlamaIndex, AutoGen, CrewAI, n8n, OpenClaw — any MCP-connected stack.

Python + TypeScript
Native SDKs for both runtimes. Framework middleware for LangChain, LlamaIndex, AutoGen, CrewAI, n8n, and OpenClaw ships Q2 2026.
Configurable policy
Built-in policies for "enterprise", "federal", and "strict" modes. Custom thresholds available.
Signed, immutable logs
Every session produces a cryptographically signed artifact. Your compliance team, auditor, and DoD contracting officer can all read it.
agent.py
import aiglos
from mcp import ClientSession

# Initialize your MCP session normally
session = ClientSession(server_url="...")

# One line wraps the entire session
agent = aiglos.attest(
    mcp_session=session,
    policy="federal",    # or "enterprise"
    log_path="./session.alg"
)

# Your agent runs normally
result = await agent.run(task="Summarize Q3 results")

# → T1-T36 scan on every tool call
# → HMAC-SHA256 signed log entry
# → PolicyViolation raised on threats

# Inspect the session artifact
manifest = agent.manifest()
print(manifest.summary())
# Calls: 47 Blocked: 2 Warnings: 1
# NDAA §1513: COMPLIANT Signed: ✓
