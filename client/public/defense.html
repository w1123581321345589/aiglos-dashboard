The Threat Reality

Your agents have root access.
Adversaries know it.

Every MCP tool call is an instruction your AI sends to your infrastructure. Without inspection, a single compromised prompt can exfiltrate credentials, pivot across systems, or corrupt data at machine speed — no human in the loop.

T-09
Credential Extraction
Agents instructed to read .env files, SSH keys, and secrets vaults via file-read tool calls.
T-17
Prompt Injection
Malicious payloads embedded in documents, emails, or web content that hijack agent behavior mid-task.
T-22
SSRF / Network Pivot
HTTP tool calls directed at internal services, metadata endpoints, or lateral-movement targets inside a secured network.
T-28
Supply Chain Injection
Compromised MCP server packages that install malicious rule overrides or exfiltration hooks at runtime.
T-31
Shell Abuse
Bash and subprocess tool calls used to establish persistence, escalate privileges, or disable monitoring.
T-36
Data Exfiltration
Outbound tool calls — writes, uploads, API calls — carrying sensitive data to attacker-controlled endpoints.
